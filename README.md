# API Data Pipeline

## Overview
This project extracts real-time data from an API and stores it in a database.

## Process
- Extract: Fetch data from API
- Transform: Parse JSON data
- Load: Save into SQLite database


## Tools Used
- Python (requests, pandas)
- SQLite
